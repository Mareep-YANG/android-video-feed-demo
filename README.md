# 项目需求
仿照今日头条的播放页（通过今日头条app首页 -> 点击视频卡进入），实现简易版视频播放流，并完成以下功能：
- 基本功能：
  - 页面交互：全屏播放页、可上下滑切换视频、数据分页拉取
  - 播控能力：暂停、播放、进度条、调节视频进度
  - 互动能力：具备 头像、点赞、评论、收藏、分享、作者名称、标题 等基本元素，但不需要响应点击事件
- 进阶功能：
  - 优化起播速度：预加载、预渲染等，手段不限，但需要通过数据阐述优化幅度。
  - 清晰度切换、横屏播放
  - 视频流内能出现其他卡片，比如图片卡。
- 补充说明：
  - 视频播放可以使用三方库或者Android/iOS系统播放器。
  - 数据分页部分可以在本地编写假数据。（视频播放链接自行在网络搜索收集）
